# Taller1 - Aplicaciones Distribuidas

Proyecto introductorio sobre conceptos básicos de aplicaciones distribuidas.

## 📚 Contenido
- Conceptos de POO: clases, objetos, métodos, atributos
- Modularidad y reutilización
- Tipos de arquitecturas: monolítica, cliente/servidor y N capas

## ⚙️ Funcionalidad
Se desarrolló una calculadora básica con JavaFX que permite:
- Suma
- Resta
- Multiplicación
- División

Todas las operaciones se ejecutan de manera local dentro de la aplicación.

## 🖥️ Interfaz
Incluye una interfaz gráfica donde el usuario ingresa datos y visualiza el resultado.

## 🚀 Ejecución
1. Ejecutar la aplicación
2. Ingresar los números
3. Seleccionar la operación
